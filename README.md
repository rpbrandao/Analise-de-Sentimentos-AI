# Azure Speech & Language Studio Lab

## 📌 Sobre o projeto
Este repositório documenta a prática com as ferramentas Azure Speech Studio e Azure Language Studio, com foco na criação de soluções baseadas em inteligência artificial para análise de fala e linguagem natural.

O objetivo foi explorar aplicações reais de IA, entender suas capacidades e identificar oportunidades de uso em cenários de negócio.

---

## 🎯 Objetivos do laboratório

- Aplicar conceitos de IA em um ambiente prático
- Explorar ferramentas de Speech-to-Text e NLP
- Documentar processos técnicos de forma estruturada
- Desenvolver visão de aplicação de IA em problemas reais

---

## 🧠 Tecnologias utilizadas

- Azure Speech Studio
- Azure Language Studio
- NLP (Processamento de Linguagem Natural)
- Speech-to-Text
- Análise de Sentimentos
- Extração de entidades
